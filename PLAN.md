# Planning Document

## Feature: Meal Planner

### MVP

A table two columns, date and meal. Date column starts at today and is a week
long.

User selects the meals from a text field with autocomplete. Can be a recipe
from the list or other meals like "Take Away".

Below, another table with all recipes and additional meals. Includes a date
last eaten, and eaten count.

Below that, ability to add additional meals that are not recipes.

### Additional Features

Each meal has labels which can be used to filter the main table. Things like:

* carb::pasta
* meat::chicken
* time::quick
* toddler::approved
